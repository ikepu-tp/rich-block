# Rich Block

This library is an editor for rich block like WYSIWYG but this is NOT WYSIWYG.

## Components

You can see components with StoryBook. [See](https://ikepu-tp.github.io/rich-block)

## How to use

### Installation

```bash
npm install @ikepu-tp/rich-block
```
